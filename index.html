API

Canvas

  • canvas.point(x, y) - cria um ponto em (x,y)
  • canvas.line(x1, y1, x2, y2, color) - cria uma linha entre (x1,y1) e (x2,y2) e colore com o RGB color
  • canvas.circle(x, y, radius, color) - cria uma circulo em (x,y) com raio radius e colore com o RGB color
  • canvas.center(x, y) - centraliza o canvas em (x,y) ou no (0,0) se (x,y) não forem informados.
  • canvas.reset() - limpa o canvas, recarrega o código e move o cursor dos pontos de teste para o começo

Eventos

  • events.on('nomeDoEvento', function) - seta um callback para o evento escolhido.
    Os eventos são:
    1. on('next', function(x, y)) - invocado quando o próximo ponto é carregado. Envia as coordenadas X e Y pelos argumentos da função.
    2. on('previous', function(x, y)) - invocado quando o ponto anterior é carregado. Envia as coordenadas X e Y pelos argumentos da função.
  • events.call('nomeDoEvento', function(err, fn)) - retorna as funções atribuidas no evento. Como o evento pode guardar múltiplas funções, ao usar este método, o usuário deve invocar o argumento fn para invocar as funções.
  • events.reset() - reseta todas funções atribuidas nos eventos.
